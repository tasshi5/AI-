# AI Knowledge Repository

このリポジトリは、生成AI（ChatGPT이나 Claudeなど）に自社の研修資料を理解させ、実務で使えるアウトプットを生成するための「AI専用ナレッジベース」です。

## 目的
- スライド資料（PDF）の内容をAIが正しく解釈できるテキスト形式で提供する
- 研修で学んだ知識をベースに、AIが「提案書」や「営業トーク」などの実践的なドキュメントを自動生成できるようにする

## ディレクトリ構成

- **`00_original/`**: 元となるスライド資料（PDF・PPT）
- **`10_text/`**: AIに読み込ませるためのテキスト化された資料群（コンテキスト）
- **`20_prompts/`**: AIに指示を出すための実践用プロンプトテンプレート

---

## 🚀 このリポジトリの使い方（AIとの連携手順）

生成AIを使って、この資料をもとに提案書などを自動作成する手順です。

### 1. 前提知識（コンテキスト）のインプット
まず、AIに「あなたがどのような知識を持っているべきか」を教えます。
- 業務特化のAIアシスタントを作りたい場合は、`20_prompts/03_system_prompt.md` の内容をChatGPTの「Custom Instructions」やプロンプトの最初に貼り付けます。
- 続いて、`10_text` フォルダ内のファイル群（特に `index.md` で全体像を把握させた後、各Lessonファイル）をAIに読み込ませます。（※一括で読み込ませるか、必要な回の内容をコピペしてください）

### 2. プロンプト（指示）の実行
AIに知識が入ったら、`20_prompts` にあるテンプレートを使って目的のアウトプットを生成させます。
- **提案書を作りたい場合**: `01_proposal_generation.md` の入力欄を埋めてAIに送信します。
- **営業トークを作りたい場合**: `02_sales_talk_generation.md` の入力欄を埋めてAIに送信します。

### 3. 出力の調整（壁打ち）
AIが出してきた回答に対して、「もう少し専門用語を減らして」「この事例をもっとプッシュして」など、対話を通じてブラッシュアップを行ってください。
