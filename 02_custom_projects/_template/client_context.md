# カスタマイズ要件定義書（クライアント・コンテキスト）

ジェネレーター（`03_generators`）のAIプロンプトを使用する際、このファイルの内容をAIに読み込ませることで、顧客に最適化された研修資料やプロンプトを出力させます。
